#  Sistema de Inventário RPG (v1.1)

Este é o meu primeiro projeto autoral em Python, desenvolvido enquanto estudo os fundamentos da programação para me tornar um desenvolvedor **Fullstack**.

## 🎯 Objetivo
O sistema simula o inventário de um personagem de RPG, permitindo que o usuário insira o nome de um item e a sua respectiva quantidade.

## 💻 Tecnologias Utilizadas
* **Python 3**: Linguagem principal. 
* **VS Code**: Ambiente de desenvolvimento.
* **Git/GitHub**: Controle de versão e portfólio.

## 🧠 O que aprendi neste código:
* **Entrada de Dados**: Uso do `input()` para interagir com o usuário. 
* **Variáveis**: Armazenamento de informações como objetos (Strings). 
* **Formatação (f-strings)**: Uso de chaves `{}` para inserir variáveis diretamente no texto, deixando o código mais limpo e visual.
* **Operadores de String**: Uso do `'-' * 20` para criar divisórias visuais e deixar a interface mais organizada.
